# 🛒 쇼핑 리스트 앱

로컬스토리지 기반의 심플한 쇼핑 리스트 웹 앱입니다. 별도 설치나 서버 없이 HTML 파일 하나만으로 동작합니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 구매 완료 표시
- 아이템 개별 삭제
- 완료된 항목 일괄 삭제
- 총 개수 / 완료 개수 통계 표시
- 페이지를 닫아도 데이터 유지 (localStorage)

## 사용 방법

1. `shopping-list.html` 파일을 다운로드합니다.
2. 브라우저에서 파일을 엽니다.
3. 입력창에 아이템을 입력하고 `+` 버튼 또는 `Enter`를 누릅니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
